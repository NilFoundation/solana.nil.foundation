About

Solana claims to be the most performant public fault-tolerant database replication protocol. It uses Berkley Packet Filter as a virtualization environment along with VDF-based cluster commit log validity checking.

Solana public cluster achieves about ~3k tx/sec in a day-to-day operation mode, up to ~50k tx/sec in 4-byte "transactions" benchmarking mode along with up to ~10k tx/sec in regular "transfers" performance benchmarking mode.
